# Sağlıktan - Sağlık Sosyal Platformu

Sağlıktan, sağlık profesyonelleri ve kullanıcıları bir araya getiren modern bir sağlık sosyal platformudur. Platform, sağlık bilgisi paylaşımı, hasta-doktor iletişimi ve sağlık takibi gibi özellikler sunmaktadır.

## Özellikler

- Modern ve kullanıcı dostu arayüz
- Responsive tasarım (mobil ve masaüstü uyumlu)
- Doktor ve kullanıcı profilleri
- Güvenli kimlik doğrulama sistemi
- Sağlık durumu takibi
- Randevu yönetimi
- Mesajlaşma sistemi

## Teknolojiler

- React 18
- Material-UI (MUI)
- React Router
- Vite
- Date-fns
- Recharts

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/yourusername/sagliktan.git
cd sagliktan
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

4. Tarayıcınızda [http://localhost:5173](http://localhost:5173) adresini açın.

## Proje Yapısı

```
src/
  ├── pages/          # Sayfa bileşenleri
  ├── components/     # Yeniden kullanılabilir bileşenler
  ├── assets/         # Statik dosyalar
  ├── styles/         # Stil dosyaları
  └── utils/          # Yardımcı fonksiyonlar
```
